The ChatApp

Description

This is a video chat application that lets two clients connect and share video and audio tracks between each other. There is also a disable video and disable audi options. In addition it includes a real-time chat with emojis. A room is only created when the first person joins and then the second person can join already existing room, when bouth parties leave the room it gets deleted.

Technicalities

Application front-end is written using React and is hosted on Netlify. The back-end is written using NodeJS and is hosted separately on Google Cloud Platform. To establish the connection the app uses HTTP long polling protocol provided by Socket.IO. Then it uses WebRTC to send and recieve video/audio data. The chat functionality is also powered by Socket.IO.

Overview

Although there are plenty of similar commercial applications available, it was interesting to recreate somehting like that, especially when everybody uses a similar application for their work needs. While working on this project I learned alot about back-end development and web protocols, as well as generally establishing connection between two clients.
